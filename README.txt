O projeto usa uma API do google maps cuja chave pode ser alterada em vitalpaint/graddle.properties, no campo google_maps_key.

É necessário fazer Login com uma conta local.
Para iniciar um jogo é preciso haver no minimo 2 jogadores em equipas diferentes, e como tal, deveram ser criadas 2 contas para 
se usar em 2 dispositivos (nao emulados)

A aplicação requer conexão permanente a internet e GPS ativado
